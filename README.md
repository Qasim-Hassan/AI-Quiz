# AI-Quiz

## Introduction

Simple quiz app where a user can attempt quiz and at the end they can get AI explaination for the questions they attempted.

----

## Tech Stack

- Vanilla JS (Frontend)
- Node JS (Backend)

----

## Setup on your machine

Make sure you have Node JS installed on your machine.
Clone the repository, cd into backend directory.
In the *api.js* file, paste your API_KEY as the value of the constant "api". Export this constant.

Then run:

> node server.js

Open the *localhost:3000* on your browser to see the app running.
